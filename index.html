Welcome to the RiskBot Demo Page

This is a demo of a risk and crisis chatbot. It uses 'Beyond The Spreadsheet', The 'ERM System Build Guide', and a model crisis management plan (CMP) by Andrew Sheves as knowledge base references to answer questions.

If it can't answer using the knowledge base, it sends the query to ChatGPT. The embedded bot can be temperamental while in development so if it doesn't pop up, you can access a browser version below.
